# MTR Maintenance Tracking - User Guide

## Schedule
- **7-day calendar view** of all maintenance slots across MTR zones (MTR-01 to MTR-06)
- **Drag-and-drop rescheduling** to move maintenance between days and time slots (23:00, 01:30, 03:30)
- **Automatic slot swapping** when moving cards to occupied slots
- **23:00 slot eligibility** indicated by clock icon - only eligible units can use this slot

## Work Order Tracking
- **Validation workflow** for past maintenance - Every morning, mark as completed or flag for rescheduling
- **Bulk validation** to process multiple work orders at once
- **Rescheduling interface** with visual slot picker showing available dates before deadline
- **Completion tracking** with status badges (Pending, Completed, Rescheduling)
- **Search functionality** to find work orders by WO number across all tabs

## Dashboard
- **Completion metrics** showing % of maintenance completed before due date
- **On-time performance** tracking completion within 6 days of MTR start date
- **Planning deviation** analysis comparing WM start date vs MTR start date
- **Execution deviation** tracking actual completion vs planned dates
- **Trend analysis** by month, batch, or zone with expandable detail views

## Admin Panel
- **Device management** - configure Zone, Week Batch, and 23:00 slot eligibility for all MTR equipment
- **Work order upload** - CSV import with automatic slot distribution and validation
- **Engineer assignments** - assign engineers to zones with fixed engineer support (one per zone)
- **Mapping requirements** - only devices with Zone + Week Batch can be linked to Work Orders

## Key Features
- **Real-time updates** - schedule changes save automatically
- **Status tracking** - Planned, Pending, Completed, Completed Late, Rescheduling, Missed
- **Looker integration** - automatic sync of engineers, devices, and maintenance visits
- **Compliance monitoring** - track adherence to 14-day cycles and operational tolerance windows
